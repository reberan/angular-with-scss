4. Local Services

4.1 Basic Services

The local services containing the necessary database queries can now be developed on the basis of the repository functionality.

First the service interfaces are defined. In addition to the standard methods, the ZooService should also return a list of zoos for a specific country. 

 interface IAnimalService 
 interface IZooService

In our example, little additional functionality is offered for the AbstractBaseEntityService. Of course, this also means that the implementation classes are correspondingly simple. The implementation of the additional services takes place using QueryDSL and predicates (see Database Queries) to deliver the desired result set. In addition to the standard methods listed in IBaseEntityService (NEW: AbstractBaseEntityService), the AnimalService should also be able to return a list of animals for a specific species. 

 class AnimalService 
 class ZooService
