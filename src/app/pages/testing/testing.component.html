8. Testing

8.1 Unit test

For general testing, see Tips&Tricks.

The updating of the schema is controlled in the same way as the core configuration. However, the test-config-wms-database.xml file is relevant here. The value for hibernate.hbm2ddl.auto should be set to at least update. To ensure that both the schema is up-to-date and that there is no legacy data, the create value is optimal.

The test is run with DB user itest. This is also configured in the test-config-wms-database.xml file.

The beans for the services must also be made available through configuration. The procedure here is analogous to the core configuration.
