2. Entities

2.1 Database

In the following, the two entities Animal and Zoo are used throughout as an example.

These entities are created as database tables and as Java POJOs (plain-old Java objects).

A respective zoo can contain several animals and represents a one-to-many relationship.

The object-relational mapping is specified using annotations. The tables and sequences are created automatically if this is activated for Hibernate (see P&S CB - Configuration).
